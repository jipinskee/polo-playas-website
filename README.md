# Polo Playas Complete Storefront

Open `index.html` in a browser.

## Included
- Responsive storefront using the supplied product imagery
- Nine products with prices, sizes and demo inventory
- Search, filters and sorting
- Product detail modal and size guide
- Persistent shopping cart using localStorage
- Quantity and stock limits
- Promo code support (`PLAYA10`)
- Shipping, tax and total calculations
- Shipping or local pickup checkout
- Demo order confirmation
- Newsletter capture
- Demo owner dashboard at `admin.html`
- CSV export for locally stored demo orders

## Important before accepting real orders
This package is a full interactive front-end demo, not a secure production backend. Before launch, connect:
1. Shopify, Stripe Checkout or Square for real payments.
2. A database or ecommerce platform for true inventory and orders.
3. Real shipping rates, taxes, return policy and size chart.
4. The official domain, support email and legal pages.
5. Clean original product photos rather than social-media screenshots.

The current prices and inventory are placeholders and can be edited in `script.js`.
